# 台风与蝉 Typhoon & Cicada

**把台风预报翻译成每个人能懂、能行动的信息。** 公益项目，免费、无广告、不商业化。
由 **日成Risen** 发起并维护。

缘起：2026 年台风美莎克在广西停留过久，600mm+ 降雨成灾——预报明明存在，
但“预报”和“普通人的行动”之间隔着一道没人翻译的鸿沟。这个项目就是那道翻译。

## 它做什么

- **路径图**：实况 + 五家机构预报同屏对比（中国 / 日本 / 美国 / 中国台湾 / 中国香港），
  按强度着色、带风圈。
- **我会不会受灾**：位置精确到**区县级**（全国 1600+ 区县）或浏览器定位。评估是**全局**的
  ——你的位置 × 所有活跃台风与残涡，取最危险的系统定档。给出风险参考 / 风雨的开始与
  **结束**时间 / 历史对照 / 该做什么。
- **完整生命周期**：观望（远台风靠近）→ 备灾 → 影响中 → **减弱期** → 过境后。过境 ≠ 结束，
  残余强降雨单独判断（美莎克/郑州型教训）。**影响范围随强度动态收窄**：14 级时可及几百公里，
  减弱到 8 级、残涡时自动缩小，避免几百公里外的弱残涡误报。
- **“此刻”锚定真实观测**：就近机场 METAR 实测优先（数值模式在减弱尾段会高报约一个风级），
  无就近站再回退 Open-Meteo。
- **分人群清单**：城市居民（可叠加高层/低洼/商铺情境）、农户、水产、畜禽、渔船沿海、
  跨境商家、货运物流、工地工程、民宿旅游业、慢病透析人群。清单按 **人群 × 阶段 × 本地
  实际影响档位** 三维匹配——过境后按真实轻重分「外围掠过 / 明显影响 / 正面重创」三档，
  不拿“排水抢救受淹作物”去套一个田里没进水的农户。
- **历史对照（城市记忆）**：双锚点——最相似的一次 + 本地最强纪录；同城才做定量对比。
  当前对照库约 280 条（~100 台风 × 城市）。
- **AI 预报复盘**：用 AI 模型的历史预报回算“提前 N 天预报，实际差多少公里”
  （首个接入：Google DeepMind FGN / FNV3，巴威 +72h 平均偏差约 52km）。
- **分享卡片**：真实地图底图 + 位置/风圈/路径按真实地理绘制，核心信息是
  「台风强度 ≠ 你受影响的程度」——离得远时它就是一张辟谣卡。

**本应用不发布预警**，只做官方信息的转译、客观计算与历史事实陈列。
灾害预警以各级气象与应急管理部门发布为准。欢迎共建，见 [CONTRIBUTING.md](CONTRIBUTING.md)。

## 运行

```bash
python3 -m http.server 8642 -d docs   # 本地站点 http://localhost:8642
sh scripts/update.sh                  # 手动刷新全部快照（台风/METAR/FNV3）+ 提交推送
```

前端**直连**温州台风网实时接口（接口开放 CORS），打开页面即是实时数据，每 5 分钟自动刷新。
无构建步骤、无第三方依赖（Python 标准库 + 前端 CDN 引入 MapLibre）。

## 结构

```
fetcher/fetch.py         抓取器：温州台风网 API → 规范化 JSON 快照
fetcher/fetch_metar.py   全国机场 METAR 实测快照（“此刻”锚点）
fetcher/fetch_fnv3.py    FNV3 AI 预报复盘：对官方实况算路径偏差
fetcher/build_history.py 区县历史台风档案（IBTrACS）
fetcher/build_regions.py 全国省市区经纬度（一次性，行政区划变更时重跑）
docs/                    静态前端（MapLibre GL JS，深色底图），GitHub Pages 直接发布此目录
docs/data/               抓取器输出（快照 + 对照库 + 历史档案 + FNV3 复盘）
.github/workflows/       云端定时：update-data(30min) / fnv3(6h) / validate(PR 校验)
```

## 数据源

| 用途 | 来源 | 说明 |
|---|---|---|
| 主数据（实况+五机构预报） | 温州台风网 `typhoon.slt.zj.gov.cn/Api/*` | 非正式公开接口，开 CORS |
| 前期/预报雨量与风力 | Open-Meteo API | 免费、开 CORS，前端直连 |
| 「此刻」实测 | NOAA aviationweather.gov METAR | 机场实测，服务端抓成快照（无 CORS） |
| AI 预报复盘 | Google DeepMind FGN / FNV3 | 官方可脚本化下载；仅用 48h 前数据（CC BY 4.0） |
| 区县历史档案 | IBTrACS v04r01（NOAA） | 1949 年以来客观统计，纯计算可复核 |
| 备用官方源 | JMA / 中央气象台 | 均验证过，接口故障时切换 |

## 数据规范与合规

- 对照库、清单的数据规范与机器校验流水线见 [CONTRIBUTING.md](CONTRIBUTING.md)，
  CI 对每个 PR 运行 `scripts/validate_analogs.py`（红线正则 + 同城定量校验）。
- 只写**对生活的影响**（被淹/停电停水/转移/停工停课/交通中断），不写死伤与经济损失。
- AI 数据只用「关联时刻 48 小时前」的历史部分（CC BY 4.0），实时实验性数据受条款限制不公开展示。

## 路线图（未来方向）

- **误差回算排行榜**：台风结束后用最佳路径回算五机构 + AI 的 24/48/72h 预报误差。
- **双情景卡**：机构分歧大时给「快速过境 vs 停滞」两个分支——分歧本身就是信号。
- **官方预警信号接入**：等级判断挂靠官方发布，应用彻底退到「转译者」位置。
- **CLI / MCP 接口**：未来可能开放命令行工具或 MCP server，方便他人用 agent 直接调用
  本站的台风影响评估能力（"我在某地，某台风会不会影响我"）。
- WeatherNext 实时数据接入（等 Google 审批；注意 48h 实验性条款）。
